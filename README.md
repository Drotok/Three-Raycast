## Voraussetzungen

- Node.js (>= v16)
- npm (wird mit Node.js installiert)
- Vite (wird automatisch installiert)

## Installation

1. Projekt clonen oder herunterladen
2. Terminal öffnen und ins Projektverzeichnis wechseln
3. Folgende Befehle ausführen:

```bash
npm install
npx vite
```
